About Me

DisPurge is a Simple bot written in Discord.js made to allow for better control over Purging Messages in a Discord Server. When a user purges a message the users name and amount of messages purged are saved to the Bots Database.

GitHub Invite Support Notice

Features

  • Custom, Configureable Purge System
  • Configurable Guild Prefix | Default: dp.
  • Database to Track Users who Purge Messages
  • Open-Source and Community Made

Command Description
dp.help Shows you the bots help message and list of commands
dp.about Tell's you some info about the DisPurge project.
dp.invite Sends you a link to invite DisPurge to your server.
dp.uptime Shows you how long it's been since DisPurge last had a restart.
dp.purge Purge/Delete a specified amount of messages 1 - 99.
dp.prefix Set the bots prefix for your server.
dp.snipe View the last message that was deleted in the current channel.td>
dp.eval Evaluate some code (Bot Owner Only).
